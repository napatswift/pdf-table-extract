## To run

1. Create a virtual environment:
   ```bash
   python -m venv venv
   ```
2. Activate the virtual environment:
   ```bash
   source venv/bin/activate
   ```
3. Install the dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Run the script:
   ```bash
    python main.py <path_to_pdf> [-s <start_page>] [-e <end_page>]
   ```
